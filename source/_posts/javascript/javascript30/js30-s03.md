#### Playing width CSS Variables and JS

1. css 추가요소
```css
/* 가상화요소 :root - html에서 root 요소는 항상 html 요소 */
:root {
 --base: #ffc600;
 --spacing: 10px;
 --blur: 10px;
}
/*
  var(css변수) - IE지원 X, 위에서 정의한 css 변수 사용
  스크립트로 :root 요소를 변경할경우에 변수로 정한값이 변경되면서 자동변경
*/
img {
 padding: var(--spacing);
 background: var(--base);
 filter: blur(var(--blur));
}
```

2. javascript 추가요소
```javascript
//변수 inputs에 ".controls input"를 담는다.
const inputs = document.querySelectorAll(".controls input");

//suffix 변수에 input에 data-sizing 속성이 있을경우 그 값(ex: px)을 담고 아닐경우 Null 담는다
//document.documentElement = 루트요소 HTML의 root는 <html>
//루트요소에 속성 셋팅, 해당 input name속성에 값은 input value와 suffix에 담은 단위
function handleUpdate() {
  const suffix = this.dataset.sizing || '';
  document.documentElement.style.setProperty(`--${this.name}`, this.value + suffix);
}

//inputs foreach문 돌려서 input의 range나, color 속성이 변경되거나 마우스이동이 되었을 경우
//위에서 정의한 handleUpdate 기능 동작
inputs.forEach(input => input.addEventListener('change', handleUpdate));
inputs.forEach(input => input.addEventListener('mousemove', handleUpdate));
```
