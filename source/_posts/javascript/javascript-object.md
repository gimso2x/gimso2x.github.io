#### 객체 [참조](https://poiemaweb.com/js-object)

> 자바스크립트의 객체는 키(key)과 값(value)으로 구성된 프로퍼티(Property)들의 집합이다

```javascript
let obj = {
  'key' : value,  // 합쳐서 property

  'key2' : function() {
    // value가 함수일경우 일반함수와 구분하기위해 메소드
  },
}
```
