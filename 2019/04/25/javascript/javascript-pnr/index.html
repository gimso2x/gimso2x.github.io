참조페이지

기본형(Primitive Type)

각 변수 간에 기본형 데이터를 복사할 경우 데이터의 값이 복사된다.
  • null
  • undefined
  • Boolean
  • Number
  • String

참조형(Reference type)

각 변수 간에 참조형 데이터를 복사할 경우 데이터의 참조가 복사된다.
  • Array
  • Object
  • Function
